## 🚀 Install and run 🚀
###  🐧 linux
1. install python3 and git

2. type commands:
``` bash
git clone https://github.com/KSNT2008ARTUR/doox.git
cd doox
sudo chmod +x *
sudo sh install.sh
```
### 🪟 Windows
1. install python3 and add python to PATH

2. clone or download repo

3. open CMD, type:
```cmd
python setup.py build
python setup.py install
```

### 🤖 Android

1. Install termux

2. type commands:
``` sh
git clone https://github.com/KSNT2008ARTUR/doox.git
cd doox
python setup.py build
python setup.py install
```
